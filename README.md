# A-GCN
## Learning Label Correlations for Multi-Label Image Recognition with Graph Networks

### Requirements

    Pytorch 0.3.1
    Python 3.6

### Demo COCO 2014

    python3 demo_coco_gcn.py --data data/coco --image-size 448 --batch-size 32 -e --resume checkpoint/coco/coco_checkpoint.pth.tar
